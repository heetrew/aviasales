# ТЗ: Свайпер-викторина (Swiper Quiz)

Проект представляет собой интерактивную игру "Правда или Ложь", где пользователю предлагается угадывать факты, свайпая карточки. Реализовано на стеке React + TypeScript + Vite.
Доступен по адресу: [aviasales.force.msk.ru](https://aviasales.force.msk.ru)

## 🚀 Как запустить

### Локальная разработка

1. Установите зависимости:
   ```bash
   yarn install
   ```

2. Запустите проект:
   ```bash
   yarn dev
   ```

### 🐳 Docker

Проект можно запустить в изолированном контейнере:

```bash
docker build -t aviasales-react .
docker run -p 8080:80 aviasales-react
```

## 🎁 Бонус

В качестве бонуса реализована точно такая же версия приложения, но **без использования React** (Vanilla JS), чтобы продемонстрировать работу без лишнего оверхеда библиотеки. Чтобы запустить просто откройте bonus/index.html
